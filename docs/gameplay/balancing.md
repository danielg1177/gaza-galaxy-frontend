# Balancing

## Status
**No formal balancing done yet.** This document will track balance decisions as the game engine matures.

## Goals
- Fair starting positions (enforced by map generator)
- Tech progression that rewards research but doesn't snowball irreversibly
- Combat randomness that feels meaningful but not arbitrary
- Ship production rates that make games last a reasonable duration
