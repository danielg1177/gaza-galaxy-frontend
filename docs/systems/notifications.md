# Push Notifications System

## Status
**Not yet planned for implementation.** Depends on multiplayer backend.

## Overview
Push notifications via Expo Notifications, delivered when game events require player attention.

## Trigger Events
- It becomes a player's turn
- A game finishes
- A player is eliminated
- Optional: reminder after extended inactivity on an open turn

## Technology
- Expo push notifications (initial implementation)
- Delivered via Laravel backend after turn resolution
